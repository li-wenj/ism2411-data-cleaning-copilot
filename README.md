# ism2411-data-cleaning-copilot

This project cleans a messy sales dataset for ISM 2411 using Python and pandas.  
It reads `data/raw/sales_data_raw.csv`, fixes column names and text, handles missing/invalid values, and writes a cleaned file to `data/processed/sales_data_clean.csv`.

## How to Run

From the project root folder:

```bash
python src/data_cleaning.py
